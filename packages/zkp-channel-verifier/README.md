# ZKP Channel Verifier

Intermediate Proof Verifier and State Channel Proof Generator for Tokamak zkEVM.

## Quick Start

```bash
# Install dependencies
npm install

# Sync binaries from Tokamak-zk-EVM
./sync-dist.sh

# Run in development
npm start

# Build production app
npm run make
```
